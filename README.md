# QuickFiller

QuickFiller is a lightweight browser extension that instantly fills web forms with realistic fake data.  
Built for developers and QA engineers who are tired of manually typing the same test data over and over.

---

## What it does

QuickFiller detects form fields on any webpage and populates them with sensible mock data in one click.

It works on:
- Signup & login forms
- Dashboards and admin panels
- Internal tools
- Checkout and onboarding flows

No accounts. No cloud processing. Everything runs locally in your browser.

---

## Features

- One-click form filling
- Realistic fake data (names, emails, passwords, addresses, etc.)
- Context-aware field detection (based on labels and input types)
- Option to skip already-filled fields
- Customizable settings
- Lightweight and fast
